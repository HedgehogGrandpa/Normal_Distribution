Программа выдаёт случайные округлённые величины из диапазона, заданного пользователем и подсчитывает количество уникальных величин

Left border и Right border  задают границы диапазона (вещественные числа. По умолчанию 100 и 200 соответственно)
Accuracy                    задаёт точность с которой считаются величины в количествах знаков после запятой (целое число. По умолчанию 0)
Quantity                    количество величин (количество опытов) (целое число. По умолчанию 100)

**Важно: не указывать точность меньшую, чем у границ отрезков

Результат выводит в формате <Номер опыта>: <Размер>
Сохраняет этот результат в .xlsx файл. Имя файла моно указать параметром XLS File name

Также выводит графики с полученными величинами в кординатах:
[№ опыта -> Размер]
